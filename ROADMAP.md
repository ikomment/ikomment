# iKomment Roadmap

This is a living document. Items move as real usage teaches us things —
open an issue to tell us what you need most.

## Now — v1 (launched)

- Comments, Q&A, and Forums modules with live chat
- Identity: guest, email sign-in links, Google, GitHub, SSO — all owner toggles
- AI spam detection, trust levels, pre-moderation, word filters, shadow bans
- Mobile-first admin dashboard with swipe moderation and per-page overrides
- Disqus one-click import
- Email via Resend or SendGrid (owner's choice)
- Self-hosting on Cloudflare's free tier

## Phase 2 — Engagement

- Reactions/emojis on comments
- Image & GIF uploads (with image moderation)
- @mentions with notifications
- Comment search

## Phase 3 — Growth & reach

- **☁️ iKomment Cloud** — the hosted plan: we run everything, you paste two
  lines. [Join the waitlist](https://ikomment.com/waitlist)
- **🐳 Docker self-hosting** — run iKomment anywhere: AWS, Hostinger,
  DigitalOcean, your own server. Cloudflare is our default, not a cage —
  the codebase already routes data access through a platform adapter built
  for this swap.
- Import tools for more platforms (WordPress comments, FastComments)
- Multilingual widget UI
- RSS feeds; SEO-rendered comments for host-site search visibility

## Phase 4 — Community depth

- Moderator role + invitations (deputise trusted members)
- Badges, reputation & gamification
- User profiles 2.0 — bios, links, activity feeds
- Webhooks & public API
- Engagement analytics for site owners

## Explicitly not planned

- Ads, tracking, or selling data — ever
- Twitter/X or Facebook login
