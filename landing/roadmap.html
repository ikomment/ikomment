Roadmap

A living plan. Tell us what you need most by opening an issue.

✅ Now — v1, launched

  • Comments, Q&A and Forums with live chat
  • Five identity options, all owner toggles
  • AI spam detection, trust levels, full moderation suite
  • Mobile-first dashboard, Disqus one-click import
  • Self-hosting on Cloudflare's free tier

Phase 2 — Engagement

  • Reactions/emojis on comments
  • Image & GIF uploads (with image moderation)
  • @mentions with notifications
  • Comment search

Phase 3 — Growth & reach

  • ☁️ iKomment Cloud — the hosted plan. Join the waitlist
  • 🐳 Docker self-hosting — run anywhere: AWS, Hostinger, your own server. Cloudflare is our default, not a cage — the codebase already routes data access through a platform adapter built for this swap.
  • Import tools for more platforms (WordPress, FastComments)
  • Multilingual widget UI
  • RSS feeds; SEO-rendered comments for host-site search visibility

Phase 4 — Community depth

  • Moderator role + invitations
  • Badges, reputation & gamification
  • User profiles 2.0 — bios, links, activity feeds
  • Webhooks & public API
  • Engagement analytics for site owners

❌ Explicitly not planned

  • Ads, tracking, or selling data — ever
  • Twitter/X or Facebook login
